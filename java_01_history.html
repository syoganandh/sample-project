Introduction & Java History

Java was created by James Gosling at Sun Microsystems. Originally developed for simple electronic consumer goods (Project Green), it evolved into the world's most popular programming language — platform independent, object-oriented, and secure.

👨‍💻

💡Java vs C/C++: C/C++ are platform dependent (compile to .exe) and statically allocated. Java is platform independent (compiles to .class bytecode) and memory is allocated dynamically at runtime when objects are created.
