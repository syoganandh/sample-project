Java Collections Framework

Collection is an object that represents a group of other objects. The Collections Framework (java.util) provides predefined classes and interfaces to store, retrieve, manipulate, and communicate aggregate data efficiently.

📋

List

Index-based, allows duplicates & null. Maintains insertion order. ArrayList, LinkedList, Vector, Stack.

🔵

Set

No duplicates, at most one null. HashSet, LinkedHashSet, TreeSet (sorted).

📬

Queue

FIFO ordering. PriorityQueue for priority-based processing. Deque for double-ended.

🗺️

Map

Key-value pairs. No duplicate keys. HashMap, LinkedHashMap, TreeMap, Hashtable.

ArrayList — Resizable Array, Fast Retrieval
import java.util.*;

ArrayList al = new ArrayList();       // default capacity 10
ArrayList al2 = new ArrayList(20);      // specified capacity

al.add("A");
al.add("B");
al.add("C");
al.add("B");         // ✅ duplicates allowed
al.add(new Integer(10)); // ✅ heterogeneous allowed
al.add(null);         // ✅ null allowed

System.out.println(al);          // [A, B, C, B, 10, null]
System.out.println(al.size());   // 6
System.out.println(al.get(1));   // B
System.out.println(al.contains("C")); // true

al.remove("B");              // removes first "B"
al.add(1, "X");              // insert at index 1
al.set(0, "Z");              // replace at index 0

LinkedList — Double Linked List, Best for Insert/Delete
import java.util.*;

LinkedList ll = new LinkedList();
ll.add("A");
ll.add("B");
ll.add("C");
ll.add("D");
ll.add("E");
System.out.println(ll);           // [A, B, C, D, E]

// LinkedList-specific methods

ll.addLast("Y");               // [X, A, B, C, D, E, Y]

ll.removeFirst();               // removes X
ll.removeLast();                // removes Y

System.out.println(s.peek());     // B (view top, no remove)

HashSet, LinkedHashSet, TreeSet
// HASHSET — no duplicates, no order, one null
HashSet hs = new HashSet();
hs.add("A"); hs.add("B"); hs.add("C");
System.out.println(hs.add("A"));    // false — duplicate!
System.out.println(hs);            // [A, B, C] (any order)

// LINKEDHASHSET — no duplicates, maintains insertion order
LinkedHashSet lhs = new LinkedHashSet();
lhs.add("B"); lhs.add("A"); lhs.add("C");
System.out.println(lhs);           // [B, A, C] — insertion order

// TREESET — no duplicates, always SORTED, no null
TreeSet ts = new TreeSet();
ts.add("B"); ts.add("E"); ts.add("A");
ts.add("D"); ts.add("C"); ts.add("F");
System.out.println(ts);            // [A, B, C, D, E, F]

// TreeSet navigation methods
System.out.println(ts.ceiling("D"));  // D (≥ D, lowest)
System.out.println(ts.higher("D"));   // E (> D, lowest)
System.out.println(ts.floor("D"));    // D (≤ D, highest)
System.out.println(ts.lower("D"));    // C (< D, highest)
System.out.println(ts.descendingSet()); // [F,E,D,C,B,A]

PriorityQueue — Priority-Based Ordering (FIFO with priority)
import java.util.*;

PriorityQueue q = new PriorityQueue(); // default cap 11
q.offer("D"); q.offer("B");
q.offer("F"); q.offer("A");
q.offer("C"); q.offer("E");
System.out.println(q);               // internal heap structure

// peek/element — read head (no remove)
System.out.println(q.peek());        // A (null if empty)
System.out.println(q.element());     // A (exception if empty)

// poll/remove — read and remove head
System.out.println(q.poll());        // A (null if empty)
System.out.println(q.remove());      // B (exception if empty)

// ❌ null not allowed → NullPointerException
// ❌ heterogeneous not allowed → ClassCastException
// ✅ duplicates allowed
// ✅ follows sorting order (natural/Comparator)

Iterating over Collections — 4 Ways
ArrayList al = new ArrayList();
al.add("A"); al.add("B"); al.add("C");

// 1. for-each loop (simplest)
for(Object obj : al)
    System.out.println(obj);

// 2. Iterator (can remove while iterating)
Iterator it = al.iterator();
while(it.hasNext()) {
    System.out.println(it.next());
}

// 3. ListIterator (bidirectional, List only)
ListIterator lit = al.listIterator();
while(lit.hasNext())
    System.out.print(lit.next() + " ");    // forward
while(lit.hasPrevious())
    System.out.print(lit.previous() + " "); // backward

// 4. Collections utility methods
Collections.sort(al);          // sort
Collections.reverse(al);       // reverse
Collections.shuffle(al);       // random shuffle
Collections.min(al);           // minimum element
Collections.max(al);           // maximum element
Collections.frequency(al,"A"); // count of "A"
