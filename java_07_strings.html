String Manipulations

Java provides four classes for string operations — String (immutable), StringBuffer (mutable, thread-safe), StringBuilder (mutable, fast), and StringTokenizer (tokenization).

// String is IMMUTABLE — operations create new objects
String str1 = new String("Durga ");
String str2 = str1.concat("Software ");
String str3 = str2.concat("Solutions");

System.out.println(str1);  // Durga          (unchanged!)
System.out.println(str2);  // Durga Software
System.out.println(str3);  // Durga Software Solutions

// String pool — literal strings are shared
String a = "Hello";
String b = "Hello";
String c = new String("Hello");

System.out.println(a == b);       // true  (same pool ref)
System.out.println(a == c);       // false (different object)
System.out.println(a.equals(c));  // true  (same content)

// StringBuffer is MUTABLE + THREAD-SAFE
StringBuffer sb1 = new StringBuffer("Durga ");
StringBuffer sb2 = sb1.append("Software ");  // modifies sb1!
StringBuffer sb3 = sb2.append("Solutions");

System.out.println(sb1); // Durga Software Solutions
System.out.println(sb2); // Durga Software Solutions
System.out.println(sb3); // Durga Software Solutions
// Note: sb1, sb2, sb3 all point to same object!

// Other StringBuffer methods

sb.reverse();              // "olleH"
sb.replace(0, 5, "Java"); // "Java"

// StringBuilder is MUTABLE + FAST (not thread-safe)
// Same API as StringBuffer — use in single-threaded apps
StringBuilder sb = new StringBuilder("Hello");

sb.append(" Java");        // "Hello Java"
sb.insert(5, ",");          // "Hello, Java"
sb.replace(7, 11, "World"); // "Hello, World"
sb.delete(5, 7);           // "HelloWorld"
sb.reverse();               // "dlroWolleH"

System.out.println(sb);    // dlroWolleH
System.out.println(sb.charAt(0));  // d
System.out.println(sb.capacity()); // 21 (default 16 + len)

// StringTokenizer — divide string into tokens
import java.util.StringTokenizer;

// Default delimiter: space, tab, newline
StringTokenizer st = new StringTokenizer("one two three");
while(st.hasMoreTokens()) {
    System.out.println(st.nextToken());
}

// Custom delimiter
StringTokenizer st2 = new StringTokenizer("a,b,c", ",");
System.out.println(st2.countTokens()); // 3

// Modern alternative: split()
String[] parts = "a,b,c".split(",");
for(String p : parts)
    System.out.println(p);

String s = "Hello Java World";

s.length()              // 16
s.charAt(0)             // 'H'
s.indexOf("Java")       // 6
s.substring(6)          // "Java World"
s.substring(6, 10)      // "Java"
s.toUpperCase()         // "HELLO JAVA WORLD"
s.toLowerCase()         // "hello java world"
s.trim()                // removes leading/trailing spaces
s.replace("Java", "C") // "Hello C World"
s.contains("Java")      // true
s.startsWith("Hello")   // true
s.endsWith("World")     // true
s.split(" ")            // ["Hello","Java","World"]
s.isEmpty()             // false
s.equals("Hello")       // false
s.equalsIgnoreCase("hello java world") // true
💡When to use what: Use String for fixed text. Use StringBuilder for string building in single-threaded code (loops). Use StringBuffer only when thread safety is required.
